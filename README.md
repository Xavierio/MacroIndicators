# MacroIndicators — 宏观周期指标

受桥水基金全天候框架启发的综合宏观周期指标面板，适用于 **Wall Street Raider**。在游戏工具栏的 Cheats 按钮旁添加 **📊 Macro Indicators**（📊 宏观周期指标）按钮。

---

## 功能

### 核心利率仪表盘（3×3 网格）
实时显示基准利率、GDP 增长率、长期国债(10Y)、短期国债(2Y)、曲线利差、套息交易空间、个人所得税率、资本利得税率、税后套息——每项均附公式分解。

### 收益率曲线分析
- 10Y−2Y 利差（倒挂深度与持续时间）
- 曲线方向（陡峭化 / 平坦化）
- 再陡峭化速度（年化变化率）

### 增长与信贷监控
- 增长-收益率差额（GDP − 债券收益率，作为股权风险溢价近似）
- 萨姆法则衰退信号（基于GDP近似）
- 信贷压力（基准利率 − 无风险利差）

### 四阶段周期相位检测器
可视化相位条：宽松期 → 复苏期 → 紧缩期 → 衰退前期。基于利率方向 + 曲线运动 + GDP 趋势的季度历史快照自动检测。

### 央行货币政策
从游戏内 Advisory Summary 文本解析。立场分类：激进宽松 / 宽松 / 中性 / 紧缩。与周期相位交叉比对，标记背离。

### 税务分析
从 Economic Data 中提取企业所得税、个人所得税、资本利得税税率。计算税后债券收益率、税后套息利差、资本利得税优势，每项资产标注税务效率标签（[CG]资本利得 / [OI]普通所得）。

### 资产类别信号（10项资产）
宏观驱动的买入/卖出/持有建议：股指、黄金、白银、原油、玉米、小麦、比特币、以太坊、长期国债(10Y)、短期国债(2Y)。每项信号附有理由说明和当前价格及涨跌幅。

### 战略建议
综合所有指标得出的超配/回避建议，结合周期相位背景。

### 套利操作手册
每个周期阶段的策略指南（宽松期、复苏期、紧缩期、衰退前期），当前阶段高亮。

### 历史追踪
季度宏观快照自动保存至存档。支持趋势型指标（曲线方向、速度、倒挂持续时间、周期检测）。首次打开时自动从游戏内置图表历史回填。

### 双语支持
完整英文和简体中文支持，自动跟随游戏语言设置。

## 数据来源

部分指标的数据仅当特定游戏视图处于打开状态时才会被引擎广播。**数据加载一次后即被缓存**，切换视图后仍保留，直至游戏重启。

| 数据 | 需要打开的界面 | 缓存 |
|------|-------------|------|
| 税率（企业/个人/资本利得） | 市场 → 行业 → 经济数据 | 是 |
| 央行货币政策 | 市场 → 行业 → Advisory Summary（或公司概览） | 是 |
| 国债收益率/基准利率/GDP | 始终可用（allSecurities） | 无需 |
| 商品及加密货币价格 | 始终可用（allSecurities） | 无需 |

**首次使用流程：**
1. 打开 市场 → 行业 视图
2. 点击 经济数据 标签（加载税率）
3. 查看 Advisory Summary 面板（加载货币政策）
4. 点击 📊 Macro Indicators — 数据加载完毕
5. 切换任意界面 — 数据保持显示

## 安装

**手动安装：** 将 `resources/` 文件夹复制到 `<安装目录>/resources/`，保持文件夹结构，重启游戏。

**文件清单：**
- `resources/app/js/components/MacroIndicatorPanel.js`（新增）
- `resources/app/js/components/Toolbar.js`（替换，添加按钮）
- `resources/app/js/locale/macroIndicators-zh.js`（新增）

## 备注

- 增量式 mod — 仅最小化修改 Toolbar.js（一次导入+一个按钮）。
- 历史快照存储在存档 customData 的 `macroHistory` 键下，不与其他 mod 冲突。
- 所有指标基于游戏内数据计算。此为决策辅助工具，不构成投资建议。
